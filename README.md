## Getting Started

1. Clone the repository:
   ```bash
   git clone [your-repo-url]
   cd project
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev:all
   ```

4. Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
